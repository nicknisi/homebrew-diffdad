# homebrew-diffdad

Homebrew tap for [Diff Dad](https://github.com/nicknisi/diffdad).

## Install

```sh
brew tap nicknisi/diffdad
brew install dad
```

The formula in this repo is updated automatically by the [release workflow](https://github.com/nicknisi/diffdad/actions) on every tagged release of the main repo. Don't edit `Formula/dad.rb` by hand.
